# Changelog

All notable changes to `php-constant` will be documented in this file

## 1.1.0 - 2021-06-28
### Added
2 additional functions to support [Laravel Collection](https://laravel.com/docs/collections) class.
- PhpConstant::collect()
- PhpConstant::collectOptions()
